Certificate Challenge Not Available

⚠️ Platform Limitation

Why?

Client certificate authentication requires direct TLS handshake between browser and this app. Your request came through a TLS-terminating proxy (e.g., Railway edge), which means the connection is encrypted at the proxy layer, not here.

For Testing Locally:

Generate local certificates and run directly:
npm run generate:cert && npm start
Then visit https://localhost:3443/challenge/certificate

/challenge/basic (username/password) works everywhere, including on managed platforms.
